🔑 Configurar API Key de Anthropic

Tu API key se guarda solo en este navegador (localStorage). Nunca sale de tu dispositivo.

1. Ve a console.anthropic.com/settings/keys
2. Pulsa Create Key
3. Copia la key (empieza por sk-ant-...)
4. Pégala aquí abajo
⚠️ Ten en cuenta que las llamadas a la API tienen coste. El uso de esta agenda es muy ligero (unos pocos céntimos al mes).
